# Updating This Content

This directory is a snapshot from [github.com/affaan-m/everything-claude-code](https://github.com/affaan-m/everything-claude-code.git).

## Important

**DO NOT manually edit files in this directory.** Manual edits will be overwritten when content is updated from upstream.

## How to Update

Run the community update script from the repo root:

```bash
bash scripts/update-community.sh
```

This updates all community content from upstream. A GitHub Actions workflow runs weekly to check for updates and create a PR.
